# InstagramClone

A Instagram login web page clone [Accessed November 25, 2020].

### Features

- Desktop page
- Mobile page version
- Form validation
- Phone screen effect

### Tecs used

- Javascript
- Css3
- Html5
- MDB (Only to material design inputs and final alert)
